# Waip AI - 纯算法文本生成图像项目
基于原生 Python 从零实现的文本转图像生成项目，不依赖任何外部库，纯手工像素级图像处理。由11岁的科文通独立开发

---

## 一、项目介绍
Waip AI 是一款轻量级、无外部依赖、纯算法实现的文本驱动图像生成工具。
项目全程不调用 OpenCV、Pillow、PyTorch、TensorFlow 等任何第三方库，所有图像生成逻辑均由基础算法手动实现。

项目目标：通过文字描述自动生成对应图像，逐步实现图像融合、特征学习、风格生成等功能。

---

## 二、已完成模块（0.0.1 ~ 0.0.6）
以下版本均位于 main 文件夹中，功能已实现并可独立运行：

### 已实现功能
- 基础文本转图像逻辑
- 像素级图像生成
- 本地数据库读取与素材调用
- 多版本迭代优化（结构、色彩、生成速度）
- 基础图像输出功能

### 版本说明
- 0.0.1 ~ 0.0.3：基础框架搭建，初步实现图像输出
- 0.0.4 ~ 0.0.6：优化生成结构、色彩表现、文件读取逻辑

所有已发布版本均可独立运行，无外部依赖。

## 四、当前存在问题（急需优化）
0.0.7 训练模块与 Database 2.0 目前存在以下严重问题：
1. 代码运行报错频繁
2. 生成图像出现**大量成群结队的噪点**，噪点互相粘连，无法实现噪点分离
3. 无法有效学习训练图片的**图像边缘轮廓**
4. 难以精准学习并还原图片的**真实颜色**
5. 算法结构尚未完全稳定
